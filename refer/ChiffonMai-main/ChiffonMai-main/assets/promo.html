📋 详细流程说明：
① 进入舞萌DX机台的「信息」页面，点击「QR 码发行」，使用微信扫描二维码；
② 在微信打开的页面中，复制以 SGWCMAID 开头的完整字符串；
③ 打开 ChiffonMai → 「同步成绩」，粘贴字符串并点击同步；
④ App 自动连接 maimai-score-hub 服务，向机台 Bot 发送好友请求；
⑤ Bot 接受请求后自动抓取官方 NET 成绩数据；
⑥ 自动推送到 Diving-Fish 水鱼账号，同时更新本地 Best50 数据；
⑦ 完成后可一键导出 Best50 长图分享给好友。

Best50 查询 & 长图导出

自动计算个人 Best50 成绩，按 Rating 从高到低排列，清晰展示每首曲目的难度、达成率、Rating 等核心数据。支持多维度拟合分析（如最高 Rating 潜力估算），以及个性化模式（锁血50、AP50、FS50 等趣味排行）。一键导出精美长图，方便分享到社交平台，记录每一次进步。

导出图片拟合分析个性化锁血50AP50

成绩查询

快速查看个人完整游玩数据，支持按难度、等级、版本等多维度筛选。历史成绩以曲线图展示趋势变化，直观呈现 Rating 成长轨迹。同时支持查看单曲详细判定数据（Perfect/Great/Good/Miss 分布），深度复盘每一局表现。

水鱼落雪历史趋势判定分析

QR 码一键同步水鱼

🎮 猜歌游戏
❓

无提示猜歌

舞萌笑传之猜猜呗① — 最纯粹的猜歌模式，无任何提示信息，仅凭曲目记忆猜测歌名。适合资深玩家挑战自我，考验对舞萌DX曲库的熟悉程度。

猜猜呗1无提示
🖼️

根据部分曲绘猜歌

舞萌笑传之猜猜呗② — 随机截取曲绘的局部区域，玩家需要通过曲绘碎片猜测对应的曲目名称。截取位置随机，面积可调（25%~75%），难度随心控制。

猜猜呗2曲绘碎片
🌫️

根据模糊曲绘猜歌

舞萌笑传之猜猜呗③ — 曲绘经过高斯模糊处理，模糊程度随时间逐渐降低，玩家需尽快识别出曲绘对应的曲目。考验对曲绘细节的熟悉度，模糊度实时变化增加了紧迫感。

猜猜呗3模糊曲绘
🎵

根据歌曲片段猜歌

舞萌笑传之猜猜呗④ — 随机播放歌曲的截取片段（时长可调 3~15 秒），玩家凭借旋律来判断曲目名称。音频片段从歌曲不同位置截取，同一首歌可能以不同段落出现，考验对歌曲的全面熟悉程度。

猜猜呗4旋律识别音频片段
🔤

根据别名猜歌

舞萌笑传之猜猜呗⑤ — 展示舞萌DX曲目的玩家常用别名或昵称，需猜测对应的官方曲名。别名数据库持续更新，涵盖国服/日服社区常见称呼，适合了解社区文化的玩家。

猜猜呗5别名挑战
🔠

舞萌开字母猜歌

舞萌笑传之猜猜呗⑥ — 曲名以空格分割为若干组，每组逐个揭示字母，玩家需在完全揭示前猜出曲名。揭示速度可调，支持自定义字母组数，互动性与竞技性兼具。

猜猜呗6逐字符揭示
👥

多人猜歌对战

支持与好友进行多人实时猜歌对战，所有 6 种猜歌模式均可游玩。基于 WebSocket 实现低延迟实时通信，支持创建/加入房间、实时同步答题状态、计分板排名。每轮结束后自动进入下一题，比拼猜歌速度和准确率，增加互动性与竞技乐趣。

WebSocket实时对战创建房间

单曲 Rating 计算

输入单曲的难度、达成率、定数等数据，精准计算该曲目对应的 Rating 分数。支持 BASIC/ADVANCED/EXPERT/MASTER/Re:MASTER 全难度计算，帮助玩家预估推分收益，合理规划冲分路线。

Rating计算推分预估
📉

达成率计算

根据 Perfect（含彩P/白P）、Great、Good、Miss 等判定详情，精准核算单曲达成率。支持反向输入 — 输入目标达成率，反推各判定数量的分配方案。适合赛后复盘，分析失分原因，针对性提升。

达成率判定核算
🔍

达成率反推

根据已知的达成率，反向推算绝赞（BREAK）判定的 Perfect/Great/Good/Miss 分布情况。输入达成率和总音符数，系统自动计算可能的判定组合，辅助复盘游玩表现，找出可优化的绝赞段落。

绝赞反推复盘分析
🔄

版本对照

清晰对照舞萌DX各版本名称与对应版本号，包括国服和日服的命名差异。涵盖从 maimai GreeN 到最新版本的完整对应关系，提供快速检索功能，方便玩家查阅版本更迭历史。

版本对照国服/日服

随机选曲

随机生成 1~4 首曲目，支持按难度、等级、版本、分类等多维度筛选。打破固定游玩习惯，发现被忽略的好歌。支持「再来一次」重新随机，适合跟朋友一起决定打什么歌。

1-4首随机多维度筛选

基于标签推荐

根据谱面难度、音乐风格、BPM 范围、版本等标签进行智能匹配，推荐符合个人喜好的曲目。推荐算法考虑玩家历史游玩数据，越用越精准。支持一键查看推荐曲目的详情和谱面信息。

智能推荐个性化

🏗️ 数据流架构

了解 ChiffonMai 如何安全高效地处理数据

📱 本地层 (ChiffonMai App)
  ├─ 本地存储（SharedPreferences）
  │   ├─ QQ 号 / 授权码缓存
  │   ├─ 排行榜参加设置
  │   ├─ 收藏夹数据
  │   └─ 应用配置偏好
  ├─ 本地计算引擎
  │   ├─ Rating 计算器
  │   ├─ 达成率核算/反推
  │   ├─ Best50 拟合分析
  │   └─ 猜歌游戏逻辑
  └─ UI 渲染层（Flutter Widget 树）

🌐 网络层 (RESTful API / WebSocket)
  ├─ Diving-Fish 水鱼 API
  │   ├─ GET /maimaidx/prober/player/  — 查询玩家数据
  │   ├─ POST /maimaidx/prober/import/ — 导入成绩
  │   └─ GET /maimaidx/prober/music/   — 获取曲库
  ├─ 落雪咖啡屋 API
  │   ├─ GET /maimai/player/{id}       — 玩家信息
  │   ├─ GET /maimai/song/list         — 曲库列表
  │   └─ GET /maimai/collection        — 收藏品数据
  ├─ maimai-score-hub (WebSocket)
  │   └─ 实时通信：认证 → 加好友 → 抓取 → 导出
  ├─ DXRating.net API
  │   └─ GET /api/songs                — 谱面标签/别名
  └─ AWMC 服务器状态
      └─ GET /status/maimai            — 服务器运行状态

🔄 QR 同步数据流
  App ──(QR码字符串)──→ maimai-score-hub 
    ──→ 机台 Bot 加好友 
    ──→ 抓取 NET 成绩 
    ──→ 返回数据给 App 
    ──→ App 推送至 Diving-Fish 
    ──→ 更新本地 Best50

所有数据在你的手机本地处理。登录水鱼 / 落雪仅用于同步官方数据，不会收集敏感信息。所有计算逻辑均在本地完成，保障数据安全。具体来说：

• 个人游戏数据（QQ 号、授权码等）仅存储在本地 SharedPreferences 中
• 排行榜参与状态可随时关闭，关闭后你的数据不会出现在任何排行榜上
• 猜歌游戏对战仅传输答案和得分信息，不涉及个人账号数据
• App 不包含任何第三方统计 SDK（如友盟、Google Analytics 等）
• 源代码完全开源，可自行审查数据安全性
QR 码同步怎么用？步骤是什么？ ▾
在舞萌DX机台登录后的信息页面点击「QR 码发行」，使用微信扫描二维码后复制以 SGWCMAID 开头的完整字符串。打开 ChiffonMai → 点击「同步成绩」→ 粘贴字符串 → 点击「同步」，App 会自动通过 maimai-score-hub 服务完成：Bot 加好友 → 抓取 NET 官方成绩 → 推送 Diving-Fish → 更新本地 Best50 的全流程。进度条会实时显示当前所处的阶段（认证中 / 加好友中 / 抓取中 / 导出中 / 完成），全程无需手动干预。
需要登录水鱼账号吗？登录信息安全吗？ ▾
首次使用 QR 码同步功能需要登录水鱼账号。在「同步成绩」对话框中输入一次水鱼用户名和密码即可，之后 App 会自动缓存 ImportToken，后续同步无需重复输入。密码仅用于获取 ImportToken，不会明文存储，也不会上传到任何第三方服务器。查询 Best50、查看排行榜、玩猜歌游戏等基础功能均无需登录水鱼账号。
和官方 NET 有什么区别？为什么要用 ChiffonMai？ ▾
ChiffonMai 是第三方工具，相比官方 NET 提供更丰富的功能：

• Best50 长图导出 — 一键生成精美长图，方便分享到社交平台
• 6 种猜歌游戏 — 支持多人实时 WebSocket 对战，趣味性拉满
• 牌子进度可视化 — 按等级 / 谱师分类展示，一键导出进度图片
• Rating 计算器 — 精准计算单曲 Rating，辅助推分规划
• 11 种多维排行榜 — 绝赞数、物量、FC/AP 等排行，全方位对比
• 收藏品攻略 — 称号/名牌/背景/面板的全收集指南
• 谱面内嵌播放 — 支持 maidata.txt 解析与可视化播放
• KALEIDXSCOPE 门进度 — 五色门进度追踪，内置攻略
• 舞萌百科 — 术语科普、机制解读、社区黑话释义
猜歌游戏支持多人吗？怎么玩？ ▾
支持！全部 6 种猜歌模式均支持 WebSocket 多人实时对战。玩法如下：

• 创建房间：选择猜歌模式 → 创建房间 → 将房间号分享给好友
• 加入房间：输入好友分享的房间号 → 加入等待 → 房主点击开始
• 实时对战：所有玩家同步答题，每题限时，答对得分，答错不扣分
• 计分板：实时显示每位玩家的得分和排名，每轮结束后自动更新
• 结束：所有题目完成后显示最终排名，可再来一局
建议使用稳定的 Wi-Fi 或 5G 网络，以获得最佳对战体验。
为什么需要联网？离线能用吗？ ▾
ChiffonMai 的大部分功能需要联网使用，因为需要从 Diving-Fish、落雪咖啡屋等第三方服务获取歌曲数据、成绩数据和排行榜信息。以下功能在联网状态下可用：

• 所有需要查询数据的功能（Best50、成绩、排行榜、收藏品等）
• QR 码同步水鱼（需连接 maimai-score-hub 和 Diving-Fish）
• 猜歌游戏多人对战（需 WebSocket 通信）
• 检查更新、问卷反馈等

以下功能可以在弱网环境下使用（但需要提前加载数据）：
• 本地 Rating 计算器、达成率计算/反推
• 本地收藏夹管理（数据存储在本地）
• 已缓存的谱面播放（需提前查看过）

你的个人游玩数据仅在你主动同步时才会传输，且仅用于查询展示，不会泄露给第三方。
支持水鱼和落雪双数据源吗？有什么区别？ ▾
支持！ChiffonMai 同时支持 Diving-Fish 水鱼 和 落雪咖啡屋 两种数据源：

• 水鱼：输入 QQ 号即可查询，数据全面，支持 Best50、成绩、排行榜等。但需要主动通过 QR 码同步来更新成绩数据。
• 落雪：需通过授权码（OAuth 2.0）登录，数据实时性更高，除基础查询外还提供收藏品数据。授权码获取方式请查看落雪咖啡屋官网。

你可以在「刷新数据」对话框中随时切换数据源，App 会自动缓存最后一次使用的数据源，下次打开时自动使用。
什么是「拟合 Best50」和「个性化 Best50」？ ▾
拟合 Best50 是在你的全部游玩记录中，自动计算出如果按照最高 Rating 排列，理论上能达到的最佳成绩组合。它可以帮你了解自己的 Rating 上限，合理规划冲分目标。

个性化 Best50 是趣味化的 Best50 排行榜，包括：
• 锁血50 — 达成率最低的 50 首曲目（排名按达成率从低到高）
• AP50 — 达成率最高的 50 首曲目
• FS50 — Full Sync+ 完成情况最好的 50 首曲目
• 更多趣味排行持续开发中

所有模式都支持一键导出长图分享。
