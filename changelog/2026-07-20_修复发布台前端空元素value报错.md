# 修复发布台前端空元素 value 报错

## 改动原因

移除参考模式后，页面已无 `#referenceArchive`，但浏览器可能仍缓存旧版 `app.js`，执行 `$('referenceArchive').value = ...` 时出现 `Cannot read properties of null (reading 'value')`。

## 具体实现

1. `index.html` 为 `app.js` / `styles.css` 增加版本查询参数强制刷新。
2. `$()` 在元素缺失时抛出明确提示（含 Ctrl+F5），避免静默空引用。

## 期望输出

强制刷新后页面可正常加载；若资源仍不匹配，错误信息会指出缺失的元素 id。

## 实际输出

本地静态检查：当前 HTML 与 JS 的 id 一一对应，无 `referenceArchive` 引用。
