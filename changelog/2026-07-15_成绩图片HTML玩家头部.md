# 成绩图片 HTML 玩家头部

## 改动原因

用户要求图片预览窗改用 HTML 渲染，并先完成两种图片类型共用的玩家头部：姓名框毛玻璃背景、头像、DX Rating、玩家名和称号，同时支持可选分辨率、可变高度和最小 3:4 比例。

## 具体实现

- 安装 Expo SDK 54 兼容的 `react-native-webview`，使用 WebView 承载独立 HTML 画布。
- 加入 1080、1440、2160 三档输出宽度；高度可输入或增减，并限制不低于 3:4。
- 将用户提供的 Arial Black 字体和 11 张 176×40 Rating 框素材纳入移动端资源。
- 按 Rating 档位选择对应框，并依据原始数字槽位置叠加五位纯数字。
- 扩展落雪玩家契约，读取 `icon`、`name_plate`、`trophy`；头像和姓名框使用官方公共素材地址。
- 水鱼仅复用其已有的称号文本；因契约不含头像和姓名框 ID，HTML 使用明确占位，不构造假资源。
- 称号样式沿用应用内已有的普通、铜、银、金和彩虹配色。
- 增加 HTML 转义、3:4 高度约束、Rating 档位、玩家素材映射、分辨率切换和 HTML 预览测试。

## 期望输出

无论选择 Best50 还是自定义，预览窗都以 HTML 画布显示同一套玩家头部；用户可选择输出宽度并增加高度，画布始终不小于 3:4。

## 实际输出

玩家头部 HTML、分辨率与高度控制已经接入。定向单元测试、界面测试、类型检查、ESLint、差异格式检查和 Android Expo 导出均已通过；导出结果确认字体及 11 张 Rating 框全部进入资源包。Best50 成绩卡片、自定义内容、生成文件和分享仍未实现，等待后续迭代。
