# Phigros 查分器接入

## 改动原因

实现 Phigros 游戏的成绩查分基础能力（数据底座 A 期），打通 TapTap OAuth2 登录 → LeanCloud 存档拉取 → 存档解密与二进制解析 → B30(RKS=Best27+Phi3) 计算 → OSS 曲库消费 → 多游戏骨架接入。

## 具体实现

### 新增文件

1. **`src/providers/phigros-auth.ts`** — TapTap Device Code OAuth2 授权流程
   - `requestDeviceCode()`: POST `/oauth2/v1/device/code`，返回 `deviceCode` + `qrcodeUrl`
   - `pollForToken()`: 轮询 `/oauth2/v1/token`，返回 `pending`/`waiting`/`TapTapToken`
   - `exchangeSessionToken()`: MAC 认证拉 profile → LeanCloud `/users` 换 sessionToken
   - `getGameSave()`: LeanCloud `/classes/_GameSave` 拉存档元数据（summary + CDN URL）
   - `downloadSave()`: 下载加密存档 ZIP

2. **`src/domain/phigros.ts`** — 领域逻辑
   - `ByteReader`: DataView 包装的二进制读取器（小端序：u8/u16/u32/float/varshort/string）
   - `decryptAes()` / `decryptBytes()`: AES-256-CBC 解密（使用 crypto-js，key/iv 与 phiTool 一致）
   - `parseSummary()`: Summary 结构体反序列化
   - `parseGameRecord()`: GameRecord 键值格式反序列化（varshort len + bitmask + score/acc）
   - `calculateRks(定数, acc)`: `difficulty * ((acc-55)/45)^2`
   - `computeB30()`: Best27(RKS排序) + Phi3(满分按定数排序)，含推分目标 ACC 反推
   - `decodeSaveZip()`: JSZip 解压存档 → 跳过版本字节 → AES 解密 → 解析 GameRecord
   - `loadDifficultyTable()`: TSV 定数表解析

3. **`src/providers/phigros-score-provider.ts`** — 实现 `ScoreProvider` 接口
   - `PhigrosScoreProvider.login()`: 完整登录流程 → 返回 `{ mode: 'phi-session', sessionToken, playerId }`
   - `getPlayer()`: 从 LeanCloud Summary 获取昵称 + RKS
   - `getRecords()`: 加载 B30 转 ScoreRecord 数组
   - `getB30()`: 计算并缓存 B30 结果

4. **`src/providers/phigros-catalog-provider.ts`** — 实现 `CatalogProvider` 接口
   - OSS `current.json` → `catalog.json` 消费链路
   - 311 首歌曲 + 定数/曲师/谱师/画师映射
   - 曲绘/头像 URL 拼接（`getIllustrationUrl()` / `getAvatarUrl()`）

### 修改文件

5. **`src/providers/contracts.ts`** — 新增 `{ mode: 'phi-session' }` 会话类型
6. **`src/domain/game-bind-options.ts`** — 新增 `'phi-taptap'` ProviderId，Phigros 游戏 `available: true`，绑定 TapTap 云存档选项
7. **`src/domain/game-data.ts`** — 新增 `kind: 'phigros'` GamePayload 变体
8. **`src/domain/game-profile.ts`** — Phigros: `ratingDigits: 2`, `bestSections: [{ id: 'b30', size: 30 }]`
9. **`src/domain/bound-account.ts`** — 新增 `createPhigrosBoundAccount()`，`PROVIDER_TITLES` 补 `'phi-taptap'`
10. **`src/domain/provider-capabilities.ts`** — `phi-taptap` 不持久化到 SQLite 快照
11. **`src/state/session-store.ts`** — `providersForAccount()` 路由 phigros → `phigrosProviders()`；`setSession()` 支持 `phi-session` 模式；`boundFromStored()` 识别 phigros 账号
12. **`src/hooks/use-game-data.ts`** — 移除硬编码 `kind: 'unsupported'`；接入 `PhigrosScoreProvider` 实时 B30 加载；`useEffect` 更新 RKS 分数到账号列表
13. **`src/storage/secure-session-store.ts`** — `phi-taptap` provider 可持久化到 SecureStore
14. **`app/(tabs)/(overview)/index.tsx`** — `displayName()` 函数识别 `kind: 'phigros'`

### 新增依赖
