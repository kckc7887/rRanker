# Phigros 自定义分隔线标题

## 改动原因

自定义成绩图分隔线仍显示「自定义N」，且与内容间距偏紧；需要按筛选条件生成与舞萌思路一致的标题，并略加大分隔线留白。

## 具体实现

- `.section-divider` 上下 margin 调整为 `14px 0 20px`
- `buildCustomPhigrosSectionTitle`：无筛选用 `BestN`，难度/评价组合为 `AT30` / `φ30` / `AT φ30`
- 分数或 Acc 有界时写入 `titleNote`，导出 HTML 以小字括号展示
- 分页保留 `titleNote`；字体收集纳入分区标题与附注

## 期望输出

自定义导出图分隔线标题随筛选变化，分数 Acc 以小字括号附注；分隔线与成绩卡间距略增大。

## 实际输出

单元测试覆盖 Best / AT / φ / AT φ 与带 `titleNote` 的 HTML 输出，全部通过。
