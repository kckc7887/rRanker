# 2026-07-10 前端 SPA 实现

## 改动原因
创建 rRanker Demo 的纯前端单页面应用，无框架依赖，深色主题。

## 具体实现
- `demo/src/frontend/index.html`: 单文件 SPA，包含 HTML/CSS/JS
- 三个视图（首页、水鱼查分、微信爬虫）通过 JS 切换，无页面刷新
- 深色主题：背景 #121212，卡片 #1e1e1e，文本 #e0e0e0，强调色 #bb86fc
- 成绩卡片展示：封面图、歌名、艺术家、评级徽章、FC/FS 徽章、达成率、RA
- 封面图加载失败时显示 🎵 占位符
- 微信爬虫流程：警告确认 → OAuth URL → 进度轮询（500ms） → 结果展示
- 搜索过滤、RA 降序排列
- 页面关闭时通过 sendBeacon 发送 /api/shutdown

## 期望输出
- 首页显示两个可点击卡片
- 水鱼查分：输入 Token → 拉取成绩 → 展示结果
- 微信爬虫：确认 → 获取 OAuth URL → 显示进度 → 展示结果
- 所有 API 调用正确，状态管理正确
