# 姓名框横幅与内嵌素材

## 改动原因

真机预览中的姓名框被毛玻璃和裁切破坏了完整长度，与参考图的横向玩家信息条不一致；应用内打包的 Arial Black 字体和 Rating 框通过本地 URI 传给 WebView 后也没有实际显示。

## 具体实现

- 移除姓名框背景的模糊、放大裁切和毛玻璃叠层，将姓名框作为保持 720:116 比例的完整图片展示。
- 将头像覆盖在横幅左侧，并在头像右侧依次排列 DX Rating、玩家名和称号标签。
- 引入与 Expo SDK 54 匹配的 `expo-asset`，配合 `expo-file-system` 读取字体和当前 Rating 框。
- 将字体和 PNG 转为 Base64 data URI 后注入 HTML，避免 WebView 读取 `file://` 或打包资源 URI 失败。
- 增加素材加载状态、失败提示，以及 HTML 布局和素材内嵌测试。

## 期望输出

预览中的玩家信息区完整显示姓名框长图且没有毛玻璃；头像、Rating、玩家名和称号按参考图形成横向布局；Arial Black 数字字体和对应档位的 Rating 框在 Android/iOS WebView 中可见。

## 实际输出

代码已改为完整横幅布局，并通过 data URI 向 HTML 内嵌字体和 Rating 框。TypeScript 类型检查通过；35 个单元测试共 128 项全部通过；7 个 UI 测试套件共 25 项全部通过；本次改动文件无 ESLint 错误或新增警告。Android 生产导出成功，并确认导出清单包含 `ariblk.ttf` 和 `rating_base_01.png` 至 `rating_base_11.png`。
