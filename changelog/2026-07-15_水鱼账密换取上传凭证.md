# 水鱼账密登录自动换取上传凭证

## 改动原因

用户确认账号密码可换 Import-Token；要求登录后自动获取，仅 SecureStore 保存，去掉「不生成 Token」等提示。

## 具体实现

- `DivingFishAuthProvider.loginWithPassword`：登录后读 `/player/profile`，无 `import_token` 则 `PUT /player/import_token`，最终持久化为 `import-token` 会话（不存 JWT）。
- 登录页去掉网页获取 / 不生成 Token 文案；手动凭证输入改为密文「上传凭证」。
- 上传可勾选文案改为「请先用账密绑定水鱼」；游戏管理会话标签改为「已可上传」。

## 期望输出

账密绑定成功后可直接勾选上传；凭证不落 SQLite/日志。

## 实际输出

与期望一致；auth 单测覆盖「已有 token / 需生成 / cookie-jar」路径。
