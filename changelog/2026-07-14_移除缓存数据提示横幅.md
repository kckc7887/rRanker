# 移除「当前显示缓存数据」提示横幅

## 改动原因

用户反馈各页面顶部的「当前显示缓存数据」类提示无打扰价值，要求去掉所有此类提示。

## 具体实现

- 从 `QueryStateView` 移除 `isStale` 属性及黄色 stale banner UI
- 有数据时直接渲染内容，不再因缓存态额外展示横幅
- 同步清理各页面传入的 `isStale` / 未使用的 `isDataStale` 解构，并更新对应单元测试
- 数据来源状态条（`SourceStatus`）仍保留，可从标签/圆点查看 live/cache，但不属于打扰式提示

## 期望输出

使用缓存回退数据时，页面顶部不再出现「当前显示缓存数据」黄色横幅。

## 实际输出

`QueryStateView` 在 stale 数据下仅渲染业务内容；相关测试通过。
