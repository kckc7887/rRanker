# Phigros 总览课题模式标签仅显示数字

## 改动原因

用户反馈：右侧“课题模式”分数标签只应显示数字（例如 `442 = 金42` 时，标签只显示 `42`），不再包含颜色前缀。

## 具体实现

1. `apps/mobile/src/domain/phigros-challenge-theme.ts`
   - 将 `formatPhigrosChallengeBadge()` 的输出从 `金42` 改为仅返回 `42`。
2. `apps/mobile/tests/dx-rating-components.test.tsx`
   - 更新断言：右侧 badge 的 value 从 `金42` 改为 `42`。

## 期望输出

- 总览页右侧“课题模式”标签仅显示数字（不包含白/绿/蓝/红/金/彩前缀）。

## 实际输出

- 单元测试 `npm run test:unit` 全量通过。
