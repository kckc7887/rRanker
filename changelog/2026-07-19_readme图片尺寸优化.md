# README 图片尺寸优化

## 改动原因

README 中的手机截图以原始尺寸渲染，在桌面端浏览器上占用过大，影响阅读流。需要把功能截图缩小，让文字与图片的比例更协调。

## 具体实现

- 将 `README.md` 中所有功能截图从 Markdown 图片语法改为 HTML `<img>` 标签。
- 手机截图统一设置 `width="280"`。
- 顶部 B50 成绩图主图设置 `width="480"`，保持视觉冲击力但不过度撑开页面。

## 期望输出

在桌面端 GitHub 渲染时，每张手机截图宽度约为 280 CSS 像素，不会占据整栏，文字与图片并排阅读更舒适。

## 实际输出

README 中 10 张图片均已通过 `<img>` 标签控制尺寸，顶部主图 480px，功能截图 280px。
