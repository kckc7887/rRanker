# Phigros 工具箱随机歌曲

## 改动原因

舞萌工具箱已有「随机歌曲」，Phigros 侧缺少同功能入口，需要按难度、定数与游玩状态随机抽取谱面。

## 具体实现

- 在 `game-toolbox` 为 Phigros 注册 `random-charts`（`/tools/random-charts`）。
- 路由按 `activeGameId` 分支：舞萌沿用原页，Phigros 使用 `PhigrosRandomChartsScreen`。
- 复用领域模块 `filterRandomCharts` / `pickRandomCharts`；难度为 EZ/HD/IN/AT 多选，结果用 `PhigrosScoreCard` 或未游玩轻量卡。
- 新增独立偏好 KV `rranker.toolbox.phigros-random-charts.v1` 与对应 Zustand filter，避免与舞萌偏好串档。
- 从 `PhigrosFilterBar` 导出 `NeutralChip` / `LevelChip` 供难度筛选复用。

## 期望输出

Phigros 工具箱出现「随机歌曲」；可按筛选抽取 1–4 条不重复谱面；无候选或无成绩时给出明确提示；筛选设置跨会话保留。

## 实际输出

- `vitest`：`random-charts`、`phigros-random-charts-preferences`、`game-toolbox` 相关断言通过。
- `jest`：`toolbox-screen` 含 Phigros「随机歌曲」渲染断言通过。
