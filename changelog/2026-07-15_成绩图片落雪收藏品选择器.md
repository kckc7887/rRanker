# 成绩图片落雪收藏品选择器

## 改动原因

用户要求在分辨率选择上方增加样式选择，并允许分别自选头像、姓名框和称号；点击对应项目后应弹出选择器，收藏品直接读取落雪完整列表。

## 具体实现

- 在成绩图片预览页的分辨率区域上方增加“样式选择”，提供头像、姓名框、称号三个入口和当前素材预览。
- 新增分页式 Modal 选择器，支持按名称或 ID 搜索、显示当前选择、恢复玩家同步素材和失败重试。
- 选择器使用 FlatList 虚拟化，只渲染可见行；头像与姓名框复用 Expo Image 磁盘缓存和落雪素材 CDN。
- 为成绩图片新增独立落雪公共收藏品查询，仅请求 `icon`、`plate`、`trophy` 完整列表，不请求无关背景，也不加载收藏品达成条件。
- 自选头像和姓名框会替换 HTML 中的落雪 CDN 素材 ID；自选称号会同时替换名称与颜色样式。
- 自选状态仅存在于当前预览页，不修改已绑定玩家资料。
- 增加 provider 请求范围测试和预览页连续选择三类收藏品的 UI 测试。

## 期望输出

用户可在分辨率选择前打开任一收藏品选择器，从落雪完整列表搜索并选择头像、姓名框或称号，关闭弹层后 HTML 预览立即显示所选素材。
