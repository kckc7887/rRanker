# 牌子进度按谱面计并微调缺失卡

## 改动原因

进度条原先按歌曲数统计，与多难度牌子不符；缺失卡去掉了歌曲 ID，难度标签也偏大。

## 具体实现

1. `calculatePlateProgress` 的 `total` / `completed` 改为各难度谱面合计。
2. 缺失卡布局：`#id` → 曲名 → 更小的 `DifficultyBadge mini`。
3. 完成数文案改为「已完成 x / x 谱面」。

## 期望输出

进度与百分比反映谱面完成度；缺失卡可见歌曲 ID，难度标签更紧凑。

## 实际输出

上述改动已落地；相关单测通过。进度条右侧仅保留「数字 / 数字」。
