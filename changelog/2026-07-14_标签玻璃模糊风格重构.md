# 标签样式重构：从霜化渐变切换到玻璃模糊

## 改动原因

用户对 SSS/SSS+ 等标签的渐变色不满意，经过多轮 HTML 样式审查后确定采用方案 4（玻璃模糊），并加深配色。

## 具体实现

1. **安装 expo-blur**：新增 `expo-blur` 依赖，用于实现 BlurView 背景模糊效果。

2. **ScoreVisuals.tsx 重构**：
   - 新增 `BlurSpec` 接口（`bg` / `border` / `text`），定义四组配色：彩虹/金/绿/蓝
   - 新增 `BlurBadge` 组件替代 `GradientBadge`，使用 `BlurView`（intensity=18, tint=light）作为模糊底层，叠加半透明有色背景和描边
   - 带 `+` 的标签（SSS+、SS+、FC+、AP+、FS+、FDX+）保留流动动画：微弱白光微光（`SHIMMER`）横向横移，不破坏玻璃模糊质感
   - 普通标签（SSS、SS、FC、AP、SYNC、FDX）仅 Glass 无流动
   - `RateBadge` / `StatusBadge` 全部走 `BlurBadge`
   - `GradientAchievement`（达成率彩虹文字）保持 MaskedView + LinearGradient 不变

3. **配色加深**：
   - 标签底色透明度 `0.18 → 0.28`，边框 `0.35 → 0.55`
   - 达成率彩虹色加深：`#ff8a96` / `#ffc888` / `#f0e470` / `#78e8a0` / `#78c8ff` / `#a89cf8` / `#f08ade`

4. **移除**：`gradientFrost` 霜化遮罩、`glowHighlight` 内光晕、`goldBadge/goldText` 纯色块样式、`GREEN/BLUE/GOLD/FLOWING_*` 渐变色常量。

## 期望输出

标签在深色背景上呈现玻璃模糊质感，色彩清晰不刺眼，带 `+` 标签有微弱微光流动。
