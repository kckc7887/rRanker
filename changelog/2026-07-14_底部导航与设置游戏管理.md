# 底部导航与设置页游戏管理改版

## 改动原因

用户要求底部导航文案调整为「总览 · 最佳 · 成绩 · 查找 · 设置」，并将设置页收敛为单一「游戏管理」入口；进入后展示已绑定账号，右下角用添加按钮绑定水鱼账号。

## 具体实现

- `app/(tabs)/_layout.tsx`：标签文案 `B50` → `最佳`，`查歌` → `查找`。
- `b50/_layout.tsx`、`search/_layout.tsx`：对应栈标题同步修改；最佳空状态文案改为「暂无最佳成绩数据」。
- `settings/index.tsx`：仅保留「游戏管理」列表行。
- 新增 `settings/games.tsx`：展示已绑定的舞萌/水鱼账号，支持解除绑定；右下角 FAB 进入添加页。
- 新增 `settings/games-add.tsx`：承接原设置页的账密 / Import-Token 登录与验证流程。
- `settings/_layout.tsx`：为设置栈注册 `games`、`games-add` 屏。
- 个人数据导入导出从设置 UI 移出，写入 `TODO.md` 待另选入口；解除绑定时仍可选择是否一并清除个人数据。
- 更新 `tests/m3-settings.test.tsx` 覆盖游戏管理列表、FAB 与解除绑定清理逻辑。

## 期望输出

- 底部五栏文案为：总览、最佳、成绩、查找、设置。
- 设置首页只有「游戏管理」；点击后看到账号列表；右下角可添加并验证水鱼账号。

## 实际输出

- UI 路由与单测按上述实现调整；`m3-settings` 三项用例通过，覆盖账号列表、FAB 跳转与解除绑定清理顺序。
