# 关闭上传弹窗保留进度小字

## 改动原因

神秘二维码（及绑定）开始上传后离开弹窗，总览按钮小字不显示进度；再打开弹窗时进度也被清掉。根因是关闭弹窗会 abort 任务并把 phase 重置为 idle。

## 具体实现

1. `UploadDataSheet`：关闭弹窗不再 abort / 不再 `applyPhase(idle)`；仅「取消」或组件卸载时中止。
2. 用 `uploadInFlightRef` 标记进行中任务；重开时若仍在上传则恢复 `running`，不重置表单与阶段。
3. 会话拉分初始 phase 使用 `authMode: 'session'`，总览小字显示「会话拉分中」。
4. 更新单测：关闭不中止且重开仍为进行中；卸载才会中止。

## 期望输出

开始上传后关闭弹窗，总览上传按钮小字持续反映阶段；再次打开弹窗仍显示进行中，可用取消中止。

## 实际输出

同上；相关 Jest 用例通过。
