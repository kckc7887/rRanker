# 忽略本地配置与规划文件

## 改动原因

`.idea`、`.trae`、`AGENTS.md`、`ROADMAP.md` 和 `TODO.md` 属于本地开发配置或规划资料，不应继续保存在远程仓库中。

## 具体实现

- 在根目录 `.gitignore` 中加入上述目录与文件的忽略规则。
- 将已经被 Git 跟踪的对应内容从索引移除，但保留本地文件。

## 期望输出

这些目录和文件不再被 Git 跟踪，后续本地修改不会出现在提交中；提交推送后，远程仓库也不再包含它们。

## 实际输出

已添加忽略规则，并从 Git 索引中移除目标内容；本地文件保持不变。忽略规则和索引状态均通过 Git 命令验证。
