# Phigros 成绩列表

## 改动原因

为 Phigros 游戏接入成绩浏览页面，沿用舞萌成绩卡片样式，展示 B30 各曲目的得分、ACC、RKS 及评级标签。

## 具体实现

### 新增文件

1. **`src/components/phigros/PhigrosScoreCard.tsx`** — Phigros 成绩卡片
   - 左侧：曲名（1行）+ 分数（FC=蓝, Phi=金）+ 难度标签 + 评级标签
   - 右侧：ACC% 小字（上）+ RKS 大字（下），上下居中
   - 评级标签：S 粉紫、V 白、FC 蓝、Phi 金色 φ

2. **`src/hooks/use-phigros-records.ts`** — Phigros 成绩 hook
   - 从 `PhigrosScoreProvider.getB30()` 获取 Best27 数据
   - 同时加载曲库目录用于曲名映射
   - 预建搜索文档供筛选使用

### 修改文件

3. **`app/(tabs)/records/index.tsx`** — 成绩页接入 Phigros 分支
   - 新增 `PhigrosRecordsScreen`：搜索框 + `PhigrosScoreCard` FlatList
   - 筛选仅支持关键词搜索（B30 仅 27 条，无需定数/难度/版本筛选）
   - 按 RKS 降序排列
   - 搜索框占位符移除 "ID"（曲名.曲师 已包含可搜内容）

4. **`app/(tabs)/search/index.tsx`** — 搜索框占位符移除 "ID"

## 期望输出

- 切换到 Phigros 游戏 → 成绩 tab 显示 B30 卡片列表
- 卡片：曲名 + 分数（FC蓝/Phi金）+ 难度标签 + 评级标签 + ACC/RKS
- 搜索框可筛选曲名
- lint/typecheck/test 全部通过
