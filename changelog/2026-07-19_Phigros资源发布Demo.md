# Phigros 资源发布 Demo

## 改动原因

需要在 Demo 总结文档中记录 Phigros 最新资源发布验证的结论与边界，避免后续把对象存储误解为玩家数据服务或应用账号系统。

## 具体实现

- 在 Demo 归档表中新增 Phigros 资源发布验证条目。
- 补充已验证链路、参考 APK 的真实整理结果与未验证事项。
- 记录 `current.json` 最后切换、密钥不进入客户端、仅分发最新版公开资源等约束。

## 期望输出

- Demo 总结可清楚说明下载探测、参考 APK 解包整理及发布入口约定。
- 文档明确对象存储不保存玩家数据，Access Key 和 Secret Key 不进入客户端。
- 文档区分已验证内容与需要用户自行验证的真实对象存储上传。

## 实际输出

- `demo/README.md` 已新增 Phigros 资源发布 POC 章节。
- 文档记录下载地址 HTTP 200、参考 APK `3.19.0.1` 整理出 1027 个资源文件（816681939 字节）的验证结果。
- 文档明确真实对象存储上传尚未代替用户执行。
