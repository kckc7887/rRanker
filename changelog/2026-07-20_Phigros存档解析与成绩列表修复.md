# Phigros 存档解析与成绩列表修复

## 改动原因

对照 `refer/astrbot_plugin_phi/phiTool`（AstrBot Phigros 插件）后确认：云存档 ZIP 解密与 `gameRecord` 二进制解析与 phiTool 不一致，导致曲目 ID 带校验垃圾字节、无法匹配定数表，B30/成绩页列表为空。

## 具体实现

### 修改文件

1. **`apps/mobile/src/domain/phigros.ts`**
   - `decryptBytes` / `decryptAes`：按 CryptoJS `sigBytes` 截断，去掉 PKCS7 残留字节
   - `ByteReader`：正确处理 `Uint8Array` 的 `byteOffset`/`byteLength`；`getString` 同步修正
   - 新增 `getGameRecordKey()`：`varshort(len) + utf8(len-2) + 2 字节校验`，与 `PhigrosLibrary.GameRecord.read` 一致
   - `parseGameRecord`：按 body 长度前缀跳转到下一条；只解析 EZ/HD/IN/AT 四档
   - `decodeSaveZip`：以 `Uint8Array` 交给解析，避免错误的 `ArrayBuffer` 视图

2. **`apps/mobile/tests/phigros-save.test.ts`**（新增）
   - 覆盖解密 padding、GameRecord 键剥离、ZIP→B30 定数匹配

## 期望输出

- 绑定后拉云存档可正确解密并解析 `gameRecord`
- 成绩页显示 B30 列表（曲目 ID 能命中定数表）
- lint / typecheck / 测试通过
