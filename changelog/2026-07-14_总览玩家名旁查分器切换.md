# 总览玩家名旁查分器切换

## 改动原因

用户已拿到 LXNS 开发者 API，准备接入舞萌第二个查分器（落雪）。先在总览页玩家概览的玩家名旁增加点击切换入口，为后续水鱼 / 落雪成绩切换铺路。

## 具体实现

- `OverviewScreen`：玩家名与「·点击切换·」组成可点行，弹出水鱼 / 落雪选项。
- `session-store`：新增 `activeProviderId` / `setActiveProviderId` 记录当前偏好；落雪成绩 provider 尚未接入时，选落雪只提示即将接入，不改成绩数据。
- `game-bind-options`：落雪副文案改为「开发者 API 已就绪，成绩绑定接入中」。
- `TODO.md`：更新落雪查分器项为 P1，记录本轮入口与后续绑定计划。

## 期望输出

总览玩家名旁可见「·点击切换·」；点击可选水鱼（当前）或落雪（即将接入提示）；相关单测通过。

## 实际输出

- 总览玩家名旁已显示「·点击切换·」，点击弹出水鱼 / 落雪选项。
- Vitest `session-store.test.ts` 7 项通过；Jest `m3-library.test.tsx` 5 项通过（含切换入口断言）。
- 选落雪时提示即将接入，成绩数据暂不切换；`activeProviderId` 已就位供后续 provider 接入。
