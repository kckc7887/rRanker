# Phigros 总览页课题模式信息卡

## 改动原因

用户要求将 Phigros 总览玩家信息卡改为课题模式驱动的视觉方案，并明确了课题模式编码口径：`442 = 金42`。同时要求将标题文案从 `RKS` 改为 `Raking Score`，并在卡片右侧增加“课题模式”大标签。

## 具体实现

1. 在 `src/domain/phigros.ts` 新增 `parseChallengeModeRank()`，按 `level = rank // 100`、`digit = rank % 100` 解析课题模式。
2. 新增 `src/domain/phigros-challenge-theme.ts`：
   - 课题模式 0-5 对应白/绿/蓝/红/金/彩主题色；
   - 提供 `resolvePhigrosChallengeTheme()` 生成信息卡主题；
   - 提供 `formatPhigrosChallengeBadge()` 生成如 `金42` 的标签文案。
3. `src/hooks/use-game-data.ts`：
   - Phigros 玩家主分标题改为 `Raking Score`；
   - 将 `summary.challengeModeRank` 注入 `payload.challengeModeRank`。
4. `src/domain/game-data.ts`：Phigros payload 增加 `challengeModeRank` 字段。
5. `src/components/DxRatingCard.tsx`：
   - 支持 `themeOverride` 自定义主题；
   - 支持 `sideBadge` 右侧半透圆角矩形标签，标题与左侧主标题同行展示。
6. `app/(tabs)/(overview)/index.tsx`：
   - Phigros 卡片使用课题模式主题色；
   - 右侧展示“课题模式”标签及 `金42` 这类值。
7. 测试补充：
   - `tests/phigros-save.test.ts` 增加课题模式解析用例；
   - `tests/dx-rating-components.test.tsx` 增加右侧 badge 渲染用例；
   - `tests/phigros-best.test.tsx` 同步 Phigros mock 字段。

## 期望输出

1. 总览页 Phigros 信息卡标题显示 `Raking Score`。
2. 信息卡主题色按课题模式等级切换（白绿蓝红金彩）。
3. 卡片右侧显示“课题模式”与半透标签值（示例：`金42`），标签值字号略小于左侧主分数字。
4. 解析口径满足 `442 = 金42`。

## 实际输出

1. 总览页已按课题模式切换配色并展示右侧标签。
2. 课题模式解析已按 `level*100 + rank` 实现，`442` 解析为 `金42`。
3. `npm run test:unit` 通过（60 files / 282 tests）。
