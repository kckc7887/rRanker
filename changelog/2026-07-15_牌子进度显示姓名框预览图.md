# 牌子进度显示 LXNS 姓名框预览图

## 改动原因

选择器收起时仅有文字，辨识度不足。落雪公开资源提供姓名框图；水鱼无对应图床，且姓名框列表本身来自 LXNS，可用 plate id 直接拼资源 URL。

## 具体实现

1. 新增 `PlateImage`：加载 `https://assets2.lxns.net/maimai/plate/{id}.png`，磁盘缓存，失败时灰色占位。
2. 牌子进度选择卡「当前牌子」标题右侧展示预览图。
3. `docs/api-protocol.md` 补充姓名框资源约定。

## 期望输出

当前选中版本牌子时，名称右侧可见落雪姓名框图；无需水鱼/落雪玩家 API 额外请求。

## 实际输出

预览已接入选择卡右侧；资源 HEAD 校验 `plate/6104.png` 返回 200。预览图相对两行文案略下移，与大字牌子名视觉对齐。
